A simple command line game where you guess a randomly generated number between 1 and 100.

## How to Run

1. **Install Rust**: Make sure you have [Rust and Cargo installed](https://www.rust-lang.org).
2. **Run the app**:
   ```cargo run```

## How to Play
- Enter a number when prompted.
- The game will tell you if your guess is too high or too low.
- If you enter something that isn't a number, the game will ignore it and ask again.
- Match the secret number to win!
